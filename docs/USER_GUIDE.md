# User Guide - Classmate Central

## Introduction

Classmate Central is a comprehensive CRM system for educational centers. This guide will help you get started and use all features effectively.

## Getting Started

### Registration

1. Navigate to the registration page
2. Fill in your details:
   - Name
   - Email
   - Password
   - Company name
3. Click "Register"
4. Check your email for verification code
5. Enter verification code to activate your account

### First Login

1. Go to login page
2. Enter your email and password
3. You'll be automatically assigned admin role for your company

## Main Features

### Dashboard

The dashboard provides an overview of:
- Today's lessons
- Revenue statistics
- Attendance statistics
- Upcoming events

### Students Management

#### Adding a Student

1. Go to "Студенты" (Students)
2. Click "Добавить студента" (Add Student)
3. Fill in student information:
   - Name
   - Age
   - Email
   - Phone
   - Status
4. Click "Создать" (Create)

#### Managing Students

- View student details
- Edit student information
- Add notes
- View payment history
- Manage subscriptions

### Teachers Management

#### Adding a Teacher

1. Go to "Преподаватели" (Teachers)
2. Click "Добавить преподавателя" (Add Teacher)
3. Fill in teacher information
4. Assign subjects
5. Save

### Schedule

#### Creating Lessons

1. Go to "Расписание" (Schedule)
2. Click on a time slot
3. Select group or individual lesson
4. Choose teacher and students
5. Set date and time
6. Save

#### Marking Attendance

1. Open a lesson
2. Click on student
3. Select status:
   - Attended
   - Missed (with reason)
   - Excused
4. Save

### Finance

#### Recording Payments

1. Go to "Финансы" (Finance)
2. Click "Добавить транзакцию" (Add Transaction)
3. Select student
4. Enter amount
5. Choose payment method
6. Add description
7. Save

#### Viewing Reports

- Transaction history
- Student balances
- Revenue reports
- Export to PDF/Excel

### Groups

#### Creating a Group

1. Go to "Группы" (Groups)
2. Click "Добавить группу" (Add Group)
3. Fill in group details:
   - Name
   - Subject
   - Teacher
   - Schedule
4. Save

### Subscriptions

#### Managing Subscriptions

1. Go to "Абонементы" (Subscriptions)
2. View active subscriptions
3. Create new subscription
4. Freeze/unfreeze subscriptions
5. Track usage

### Roles and Permissions

#### Inviting Users

1. Go to "Роли и права" (Roles and Permissions)
2. Click "Пригласить пользователя" (Invite User)
3. Enter email
4. Select role
5. Send invitation

#### Managing Roles

- View system roles
- Assign permissions
- Create custom roles (if admin)

## Tips and Best Practices

### Student Management

- Keep student information up to date
- Add notes for important information
- Track attendance regularly
- Monitor balances

### Schedule

- Plan lessons in advance
- Set up recurring lessons
- Mark attendance immediately after lessons
- Review schedule weekly

### Finance

- Record payments promptly
- Reconcile balances regularly
- Export reports monthly
- Keep transaction descriptions clear

### Notifications

- Email notifications are sent for:
  - Payment confirmations
  - Student absences
  - Account verification

## FAQ

### How do I reset my password?

Contact your administrator to reset your password.

### Can I export data?

Yes, you can export:
- Students list (PDF/Excel)
- Transactions (PDF/Excel)
- Schedule (PDF/Excel)

### How do I mark a student absent?

1. Open the lesson
2. Click on the student
3. Select "Missed"
4. Choose reason
5. Save

### What roles are available?

- Admin: Full access
- Manager: Management functions
- Teacher: Teaching-related features
- Accountant: Finance access
- View Only: Read-only access
