# Поддержка кириллицы в PDF экспорте

Для правильного отображения кириллических символов в экспортируемых PDF файлах необходимо добавить шрифты с поддержкой UTF-8.

## Вариант 1: Использование системных шрифтов Windows

На Windows система автоматически попытается использовать Arial из `C:/Windows/Fonts/`, который поддерживает кириллицу.

## Вариант 2: Добавление шрифтов DejaVu (рекомендуется)

1. Скачайте шрифты DejaVu Sans:
   - С официального сайта: https://dejavu-fonts.github.io/
   - Или используйте пакетный менеджер вашей ОС

2. Создайте директорию `fonts/` в корне проекта backend:
   ```
   backend/
   └── fonts/
       ├── DejaVuSans.ttf
       └── DejaVuSans-Bold.ttf
   ```

3. Скопируйте файлы шрифтов в эту директорию

## Вариант 3: Использование системных шрифтов Linux

На Linux система автоматически попытается использовать шрифты из `/usr/share/fonts/truetype/dejavu/`, если они установлены.

Для установки на Ubuntu/Debian:
```bash
sudo apt-get install fonts-dejavu
```

## Проверка

После добавления шрифтов перезапустите сервер. Система автоматически обнаружит и использует шрифты для правильного отображения кириллицы в PDF.

## Примечание

Если шрифты не найдены, система будет использовать стандартные шрифты (Helvetica), которые **не поддерживают кириллицу** и могут отображать символы некорректно.
