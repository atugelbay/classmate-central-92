## Технические детали

### Хранение настроек
- UI настройки сохраняются в `localStorage` с ключом `ui-preferences`
- Настройки применяются автоматически при загрузке приложения
- Синхронизация между вкладками браузера

### CSS переменные
Динамически изменяемые переменные:
- `--primary` - основной цвет
- `--primary-light` - светлый оттенок
- `--primary-dark` - темный оттенок
- `--primary-foreground` - цвет текста на primary фоне
- `--sidebar-primary` - цвет сайдбара
- `--sidebar-ring` - цвет focus ring в сайдбаре

### Применение классов
Классы применяются на корневом элементе `<html>`:
- Размер интерфейса: `ui-{size}`
- Анимации: `reduce-motion` (если выключены)
- Плотность: `density-{density}`

## Как использовать

1. Перейдите в **Настройки** → **Интерфейс**
2. Выберите желаемую тему (светлая/темная/системная)
3. Выберите цветовую палитру из 6 вариантов
4. Настройте размер интерфейса под свои предпочтения
5. Включите или отключите анимации
6. Выберите плотность отображения данных
7. Все изменения применяются мгновенно и сохраняются автоматически

## Преимущества

✓ **Персонализация** - каждый пользователь может настроить интерфейс под себя
✓ **Доступность** - можно отключить анимации для людей с вестибулярными расстройствами
✓ **Производительность** - настройки хранятся локально, не требуют запросов к серверу
✓ **Удобство** - плотность данных позволяет показать больше информации на экране
✓ **Профессиональный вид** - 6 красивых цветовых палитр на выбор
